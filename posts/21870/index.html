2. 判断符号说明

条件表达式 含义
-eq 等于（整数）
-ne 不等于
-gt 大于
-lt 小于
-ge 大于等于
-le 小于等于
-z str 字符串是否为空
-n str 字符串是否非空
str1 = str2 字符串相等
-f file 是否为普通文件
-d dir 是否为目录

注意：[ 和 ] 要有空格；变量最好加引号防止空值。
